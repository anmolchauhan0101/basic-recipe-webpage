# 🍽️ Basic Recipe webpage

A simple and basic recipe website built using **HTML and CSS**.

## 🚀 Features

- Modern navigation bar
- Food sections (Momos & Pizza)
- Ingredients and recipe lists
- Image hover overlay effect
- Clean layout with gradient background
- Footer section

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Gradients, Hover Effects)
